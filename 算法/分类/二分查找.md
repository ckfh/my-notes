# 二分查找

## 69.x的平方根

```java
class Solution {
    public int mySqrt(int x) {
        int low = 1;
        int high = x;
        int ans = 0;
        while (low <= high) {
            int mid = (low + high) >>> 1;
            // 用平方数判断会溢出:
            int div = x / mid;
            if (mid > div) {
                high = mid - 1;
            } else if (mid < div) {
                ans = mid;
                low = mid + 1;
            } else {
                return mid;
            }
        }
        return ans;
    }
}
```
